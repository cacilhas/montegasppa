Thoughts :: Relações cromáticas e temperadas

April 23rd, 2018
A hippo in a flower field.

Aprendemos que as relações entre os graus de uma escala precisam ser números racionais para serem agradáveis, mas isso não é bem uma verdade.

Um exemplo de como isso causa erros é a sobreposição de quintas: sobrepondo as frequências de 3/2 – c, g, d, a… até fazer a volta e cair de volta no c: quando chegamos ao c sete oitavas acima não temos uma frequência 128 (27) vezes maior, mas 129,75 vezes maior.

Algo similar acontece para os intervalos de quarta: em vez de chegarmos a 32 (25) vezes, chegamos a 31,57.

Outros problemas de distorção de frequência acontecem simplesmente modulando o tom de uma música. Por exemplo, a segunda do d (9/8 × 9/8) é e, cuja relação com c deveria ser 5/4, mas o resultado é 81/64.

Além dessas inconsistências, ainda temos as diferenças cromáticas entre notas similares, como as (a♭) e gis (g♯), que diferem em uma coma pitagórica (81/80).

Todos esses problemas ocorrem nas escalas cromáticas, que são aquelas que se baseiam em intervalos racionais simples.

Intervalos temperados

Para resolver isso, a escala de intervalos temperados usa relações irracionais entre os graus. Todos os intervalos são representados na forma de 2n/12, sendo n um número inteiro entre 0 e 11, inclusive, exatamente os códigos usados para os graus no Serialismo Dodecafônico.

Essas relações irraciais fazem com que as frequências dos graus nunca divirjam.

Assim a sobreposição de quintas por sete oitavas chega exatamente a 128 (27 ou 284/12); a sobreposição de quartas por cinco oitavas chega exatamente a 32 (25 ou 260/12); a segunda da segunda (22/12 × 22/12) é exatamente a terça (24/12); e assim por diante.

As relações são todas irracionais, porém próximas o bastante de relações racionais simples para soarem agradáveis:
